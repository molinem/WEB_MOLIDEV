MoliBot

Descripción del proyecto

El desarrollo surge para cubrir una necesidad propia, monitorizar los servicios alojados en un servidor de una forma sencilla. Para poder hacer uso de este bot, es requisito indispensable tener un certificado SSL con ello las comunicaciones entre el bot y nuestro servidor serán seguras. Además el código es escalable, lo que quiere decir es que tu mismo tienes el acceso completo a poder realizar las modificaciones que consideres importantes.
Un ejemplo de funcionamiento sería el siguiente : podrías consultar el estado de un contenedor docker con un simple mensaje al bot.
