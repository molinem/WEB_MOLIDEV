Horno de tratamiento térmico

Descripción del proyecto

El proyecto surge para cubrir la necesidad de un cliente. El objetivo principal del software es la obtención de las curvas de tratamiento térmico del video-registrador que dispone el horno. Debido a que previamente el software existente era lento e ineficaz, se decide crear desde 0 una solución software de acuerdo con las necesidades del cliente. Podemos destacar el uso de Multithreading para la ejecución de tareas en paralelo y el uso de protocolos seguros como SFTP para que el tráfico siempre vaya cifrado.
