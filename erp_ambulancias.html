ERP Ambulancias

Descripción del proyecto

Cuando inicié este proyecto de diseño web, mis habilidades en HTML y CSS eran limitadas, pero pronto me adapté a frameworks como Bootstrap 4, que simplificaron significativamente el diseño.
En el ámbito de la programación, enfrenté el desafío de aprender PHP, un lenguaje que había querido dominar durante años, y lo complementé con JavaScript y jQuery, una combinación eficaz para el proyecto. Afortunadamente, mi experiencia previa facilitó la transición a lenguajes orientados a la web. En cuanto a la base de datos, manejé un sistema complejo con 23 tablas. Para garantizar la escalabilidad futura del proyecto, trabajé en la independencia y separación de las vistas del código. Uno de los mayores desafíos fue lograr que la web fuera completamente responsive, es decir, adaptable a cualquier dispositivo.
